=============================
``1-my_list`` MODULE
=============================

Using ``MyList``
---------------------

Import the class:

    >>> MyList = __import__('1-my_list').MyList

NOW WE CAN TEST IT:

    >>> my_list = MyList()

    >>> my_list
    []

    >>> my_list.append(2)

    >>> my_list.append(9)

    >>> my_list.append(7)

    >>> my_list
    [2, 9, 7]

    >>> my_list.print_sorted()
    [2, 7, 9]

    >>> my_list
    [2, 9, 7]

    >>> my_list = MyList([1, 3, 2])

    >>> my_list
    [1, 3, 2]

    >>> my_list.print_sorted()
    [1, 2, 3]

    >>> my_list = MyList("hello")
    
    >>> my_list
    ['h', 'e', 'l', 'l', 'o']

    >>> my_list.print_sorted()
    ['e', 'h', 'l', 'l', 'o']

    >>> NEW = MyList()

    >>> NEW.print_sorted()
    []
